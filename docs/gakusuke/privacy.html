ガクスケ / Gakusuke

Privacy Policy

Last updated: March 23, 2026

ガクスケ is an app that converts photographed school prints into structured schedule information, then registers that information in Google Calendar. If the AI returns a map, handout image, or similar reference image, the app may save that image to the user's Google Drive. This Privacy Policy explains how Google user data and related content are handled.

1. Information and Google scopes accessed

The app requests only the Google API permissions necessary for its core functions.

  • .../auth/calendar.app.created: used to create and manage calendar events created by ガクスケ.
  • .../auth/calendar.calendarlist.readonly: used to read available calendar list information so the user can choose where events should be added.
  • .../auth/calendar.acls: used when calendar access settings need to be checked or managed in connection with the calendar the app works with.
  • .../auth/drive.file: used to create, open, or manage only the Drive files created or selected through the app, such as saved AI-returned map images.
  • .../auth/generative-language.peruserquota and .../auth/generative-language.tuning.readonly: used for the AI analysis workflow and the user's Google AI quota / related read-only model access.

2. How data is used

  • Captured school print images are analyzed to identify schedules, events, dates, times, and related school information.
  • Extracted schedule information is used to create or update Google Calendar entries requested by the user.
  • If an AI response includes a map or reference image, that image may be saved to the user's Google Drive to support later viewing and sharing by the user.
  • Google API access is used only for these in-app functions and not for advertising or unrelated profiling.

3. What the app does not request

Based on the provided scope selection screenshot, basic identity scopes such as userinfo.email and userinfo.profile are not selected as part of this access request.

4. Data sharing

Google user data obtained through these permissions is not sold. It is not shared with unrelated third parties for advertising purposes. Data is used to operate the requested features of ガクスケ and to interact with the Google services selected by the user.

5. Data retention and deletion

  • Calendar events created by ガクスケ remain in the user's Google Calendar until the user edits or deletes them.
  • Drive files saved by ガクスケ remain in the user's Google Drive until the user deletes them.
  • The user may revoke Google account access for ガクスケ at any time from Google account security settings.

6. Security

The app is designed to limit Google data access to the minimum scopes needed for schedule creation and optional Drive image saving. Access tokens and Google API permissions are used only for authenticated requests required to provide the app's features.

7. Changes to this policy

This policy may be updated if the app's features or Google API usage change. Updated versions will be posted on this page.

If you are reviewing ガクスケ for Google API verification, this page is intended to describe the app's current OAuth scope usage reflected in the provided scope screenshot.
