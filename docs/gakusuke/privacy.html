ガクスケ / Gakusuke

Privacy Policy

Last updated: April 2, 2026

ガクスケ is an app that converts photographed school prints into structured schedule information and registers that information in Google Calendar. Images and related data generated during this process may be stored in the user's Google Drive. This Privacy Policy explains how the app accesses, uses, stores, and protects Google user data.

1. Data Accessed

The app requests only the Google API permissions necessary for its core functions:

  • calendar.app.created: create and manage events generated by the app.
  • calendar.calendarlist.readonly: display available calendars for user selection.
  • calendar.acls: check or manage calendar access settings related to app-created events.
  • drive.file: create and manage files generated by the app in the user's Google Drive.
  • generative-language.peruserquota and generative-language.tuning.readonly: perform AI processing using Google Gemini API.

2. Data Usage

  • User-provided school print images are analyzed to extract schedule-related information.
  • Extracted data is used to create calendar events in Google Calendar.
  • Images and related data may be stored in the user's Google Drive.
  • All data usage is strictly limited to app functionality and never used for advertising or profiling.

3. Data Storage

  • No developer server: The app does not store user data on any developer-operated server.
  • Google account storage: All persistent data is stored only in the user's Google account (Google Drive / Google Calendar).
  • Local tokens: Authentication tokens are stored securely on the user's device only.
  • Users retain full control and can delete their data at any time.

4. Security and AI Processing

  • Temporary local caching: Images are temporarily cached on-device only for processing and are not permanently stored locally.
  • Encrypted transmission: All communication with Google APIs is performed over HTTPS/TLS encrypted connections.
  • Direct Google API usage: The app communicates directly with Google APIs and does not route data through developer servers.
  • Third-party AI service: The app uses Google Gemini API to analyze images and extract schedule information.
  • Limited use: Data is used only for requested functionality and not for advertising or unrelated analytics.
  • No model training: The developer does not use user data to train any proprietary AI models.

5. Google User Data

Google user data is used only for core functionality:

  • Creating and managing calendar events
  • Storing app-related data in Google Drive
  • Analyzing images using Google Gemini API

Google user data is never sold or used for advertising, profiling, or unrelated analytics.

6. Data Sharing

Google user data is never sold, rented, or shared with third parties for advertising or unrelated purposes.

This app complies with the Google API Services User Data Policy .

7. Security and Revocation

Users can revoke access at any time via: Google Account Permissions.

8. Changes

This policy may be updated as features evolve.
